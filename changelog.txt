List of changes mb.20beta1.B
 1 - added cadence sensor calibration in advanced mode.
 2 - added calibration of maximum torque value (torque sensor range).
 3 - added ADC conversion factor calibration of the torque sensor (with weight).
 4 - added ADC offset adjustment of the torque sensor.
 5 - added choice to display, standard or advanced mode for cadence and torque sensors.
 6 - added data display in automatic sequence when the lights are turned on.
 7 - added other data that can be shown on the display.
 8 - access to parameters and data also with error code present.
 9 - separate definitions for TARGET_MAX_BATTERY_POWER and TARGET_MAX_BATTERY_CAPACITY.
10 - with WALK_ASSIST_DEBO UNCE_TIME, the brake sensors in addition to the motor now also stop time.
11 - CRUISE mode now also operable with walk assist command.
12 - at level 0 = OFF the accelerator is disabled.
13 - moved the parameters for error E04 "motor blocked" to the config.h file.
14 - modified " fix overrun " for a quick stop of the motor when you stop pedaling.
15 - modified the manual reset of the residual battery percentage.
16 - fixed data saving problem in eeprom.
17 - fixed display battery residual percentage problem at startup.
18 - fixed problem displaying error E06 "over temperature".
